Performs an exact in-place string replacement in a single file.

Usage:
- Always read the target file first so you can supply an exact `oldString`.
- `filePath` is required and must point to the file you want to modify.
- `oldString` must be copied exactly from the file contents. Preserve whitespace and indentation exactly.
- `newString` is the replacement text. It must be different from `oldString`.
- If the match is ambiguous, expand `oldString` with more surrounding context instead of guessing.
- Use `replaceAll: true` only when every occurrence should change.
- Prefer this tool for focused edits to existing files. Use `write` only when replacing the full file contents is the right move.
- A successful call should produce a diff. If required fields are missing, the tool call failed and no diff will exist.
