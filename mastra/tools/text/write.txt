Writes the full contents of a file.

Usage:
- `filePath` is required and must point to the file you want to create or replace.
- `content` must contain the complete final file contents, not a patch or partial snippet.
- Prefer `edit` for small changes to an existing file. Use `write` when replacing the whole file is intentional.
- If the file already exists, read it first before overwriting it.
- A successful call should produce a diff against the previous contents.
